Learn JavaScript

NodeSchool is for people totally new to programming, as well as experienced programmers looking to try a new language or skillset. Knowing some JavaScript will help, but is not necessary.

Self-Directed

NodeSchool workshops are self-guided. No boring lectures. All challenges are open source and are available online at nodeschool.io. This means that you can practice anytime.
